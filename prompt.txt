# Formula One Scrollytelling Factory

You are a senior creative director, cinematic storyteller, motion designer, frontend architect and scrollytelling engineer.

Your goal is to build a premium Formula One inspired scrollytelling website similar in quality, immersion and polish to Travel Next Level.

IMPORTANT:

This is NOT a traditional landing page.

This is NOT a website with sections and scroll animations.

This is a frame-controlled cinematic experience.

The central rule of the project is:

Scroll Progress → Frame Number → Canvas renders that exact frame.

The website must use the provided master video and existing keyframes as source assets.

Do NOT generate new story concepts.

Do NOT generate new keyframes.

Do NOT redesign the narrative.

The assets already exist.

Your mission is to transform them into a complete scrollytelling experience.

# Technical Architecture

Build the project using:

* Vite
* GSAP
* ScrollTrigger
* Lenis

The primary renderer must be CanvasSequence.

The website must:

1. Convert the provided video into frames.
2. Generate manifest.json.
3. Render frames on canvas.
4. Map scroll progress to frame number.
5. Update canvas continuously while scrolling.

The implementation must follow:

Scroll Progress
↓
Frame Index
↓
Canvas Render

Do not use the original video as the hero experience.

Use frame sequences.
